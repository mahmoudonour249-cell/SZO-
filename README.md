## What is SZO?

SZO is a personal AI operating system — not a prompt, not a chatbot persona.

It is a **22-layer cognitive framework** designed to sit on top of any AI model and transform it from a generic assistant into a structured Chief of Staff that:

- Thinks ahead of the user
- Challenges weak assumptions
- Builds a self-maintaining knowledge infrastructure
- Tracks the user's cognitive evolution over time
- Operates with persistent identity across sessions
